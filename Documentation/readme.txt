Instrucciones:
	Colors and Texture:
		Color: Color that multiplies the "Albedo (RGB)"
		Albedo (RGB): Main Texture
		Foam: Foam Texture
		Scroll X & Scroll Y: Speed at which the texture moves in a vector

	Shoreline:
		Blend Color: Shoreline color
		Soft Factor: Shoreline gradient
		fade Limit: Distance at which Shoreline is painted

	Waves:
		Frequency: Wave frequency 
		Speed: frequency speed
		Amplitude: Wave amplitude
